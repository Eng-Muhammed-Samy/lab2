# lab2
### delete remote branch (dev, test)
```
git push origin :dev
git push origin :test
```
### to delete local branch (dev, test)
```
git branch -d dev
git branch -d test
```
### to delete local branch (dev, test) before merge
```
git branch -D dev
git branch -D test
```
